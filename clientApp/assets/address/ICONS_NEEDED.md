# Address Type Icons Needed

Please add the following 3D icon images to this folder:

1. **house-3d.png** - 3D icon of a private house (recommended size: 200x200px)
2. **apartment-3d.png** - 3D icon of an apartment building (recommended size: 200x200px)
3. **government-3d.png** - 3D icon of a government building (recommended size: 200x200px)
4. **office-3d.png** - 3D icon of an office/business center (recommended size: 200x200px)

## Where to find icons:

- Generate using AI tools (DALL-E, Midjourney, etc.)
- Download from icon libraries (Flaticon, Icons8, etc.)
- Use 3D modeling tools (Blender, etc.)
